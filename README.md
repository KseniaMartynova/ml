#Сравнение методов кластеризации на основе семантических эмбеддингов

Цель
Построить семантические кластеры GO-терминов на основе эмбеддингов Hig2Vec.
Выбрать и реализовать один из двух подходов на выбор:
A) Иерархическая кластеризация
B) Понижение размерности + кластеризация (UMAP / t-SNE + DBSCAN)
В результате должна получиться таблица, где каждому GO-термину присвоен номер кластера, и визуальное
подтверждение корректности кластеризации.
